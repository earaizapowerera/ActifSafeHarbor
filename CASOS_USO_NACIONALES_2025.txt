PUNTOS DE REVISIÓN CLAVE (PARA TODOS LOS CASOS):
===================================================================================

A. CÁLCULOS FISCALES (Columnas FI_):
   1. ✓ INPC Adquisición: Debe corresponder al mes de FECHA_COMPRA
   2. ✓ INPC Mitad Ejercicio: Debe ser INPC de junio 2025 o mes medio si baja
   3. ✓ Factor Actualización P1: INPC_Mitad_Ejercicio / INPC_Compra
   4. ✓ Saldo Actualizado P1: Saldo_Inicio_Año * Factor_P1
   5. ✓ INPC Mitad Periodo: Mes medio correcto (baja o junio)
   6. ✓ Factor Actualización P2: INPC_Mitad_Periodo / INPC_Compra
   7. ✓ Deprec Fiscal Actualizada: Dep_Ejercicio * Factor_P2
   8. ✓ 50% Deprec Fiscal: Dep_Actualizada * 0.5
   9. ✓ Valor Promedio: Saldo_Actualizado - 50% Dep
  10. ✓ Valor Prom. Proporcional: Valor_Promedio * (Meses_Uso_Ejercicio/12)

B. CÁLCULOS SAFE HARBOR (Columnas SH_):
   1. ✓ INPC Junio: Siempre INPC de junio 2025 (sin mes medio)
   2. ✓ Factor SH: INPC_Junio / INPC_Compra
   3. ✓ Saldo SH Actualizado: Saldo_Inicio_Año * Factor_SH
   4. ✓ Dep SH Actualizada: Dep_Ejercicio * Factor_SH
   5. ✓ 50% Dep SH: Dep_SH * 0.5
   6. ✓ Valor SH Promedio: Saldo_SH - 50% Dep_SH
   7. ✓ Valor SH Proporcional: Valor_SH * (Meses_Uso_Ejercicio/12)
   8. ✓ Prueba 10% MOI: 10% del MOI original
   9. ✓ Valor Reportable: MAX(Valor_SH_Proporcional, Prueba_10%)
  10. ✓ Saldo Fiscal Histórico: Correcto
  11. ✓ Saldo Fiscal Actualizado: Correcto

C. FECHAS:
   1. ✓ Fecha Adquisición: Debe estar en Staging_Activo.FECHA_COMPRA
   2. ✓ Fecha Inicio Depreciación: Calculada correctamente
   3. ✓ Fecha Fin Depreciación: FECHA_INICIO + (1/Tasa_Mensual - 1) meses
   4. ✓ Fecha Baja: NULL o fecha válida

D. VALIDACIONES ESPECIALES:
   1. ✓ Activos depreciados: Valor Reportable debe ser 0 o Prueba 10%
   2. ✓ Activos nuevos: Proporción correcta por meses de uso
   3. ✓ INPCs: No deben ser NULL (excepto si hay error documentado)
   4. ✓ MOI: Debe coincidir con el costo histórico
   5. ✓ Tasas: 5%, 8%, 10%, 15%, 25%, 30% son válidas
